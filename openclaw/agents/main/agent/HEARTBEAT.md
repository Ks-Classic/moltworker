# HEARTBEAT.md — 定期実行タスク

## セッション開始チェック

ハートビート受信時・セッション開始時に順番に確認：

- [ ] `memory/YYYY-MM-DD.md` が今日分存在するか？なければ作成
- [ ] 未完了タスクがあれば報告
- [ ] 定期レポートの期限を確認

## デイリーブリーフィング

毎セッション開始時、以下のソースから情報を取得してブリーフィングを生成：

1. **Google Calendar** — 今日の予定（gog スキル）
2. **Lark** — オープンな商談・期限が近いアクション
3. **GitHub** — オープンPR・アサインされたIssue
4. **Chatwork** — おさる式マーケティングルームの重要メッセージ
5. **Google Sheets** — 管理シートの更新確認
6. **memory/** — 昨日の「明日やる」項目、未完了タスク

## 静時間

深夜（23:00-08:00 JST）は緊急でなければ静かにしろ。

## 何もなければ

`HEARTBEAT_OK` を返す。
